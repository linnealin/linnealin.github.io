# DT thorax utan kontrast

DT thorax utan kontrast ger oftast mer information än lungröntgen, men innebär en högre stråldos för patienten.

::: warning Indikation
1. Pneumothorax
2. Pneumoni
3. Pleuravätska
4. Stas
5. Lungmalignitet
6. Covid-19-förändringar
:::

::: tip Normalfynd
Inga infiltrat. Ingen pleuravätska. Ingen stas.
:::
